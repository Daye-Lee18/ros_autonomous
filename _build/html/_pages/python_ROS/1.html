9. Python Library for ROS#

9.1. rclpy#

rclpy는 ROS2 python client library이다.

9.1.1. rclpy.init()#

rclpy.init()는 rclpy를 초기화하여 ROS2 네트워크와 통신할 수 있도록 준비하는 필수 함수이다. 노드 생성 전 가장 먼저 호출되어 노드 초기화 및 통신 환경 설정을 수행한다.

  • ROS2 네트워크 연결: 노드를 ROS2 시스템 (Graph)에 등록하고, 다른 노드들과의 통신(Topic, Service, Action 등)을 준비한다.

  • 초기 설정: args인자를 전달받아 실행 시 필요한 옵션을 설정할 수 있다.

import rclpy

def main(args=None):
    # 1. rclpy 초기화 (필수)
    rclpy.init(args=args)
    
    # 2. 노드 생성 및 처리 (생략)
    # node = MyNode()
    # rclpy.spin(node)
    
    # 3. 종료 시 rclpy.shutdown() 호출
    rclpy.shutdown()

9.1.2. Node 관련#

9.1.2.1. rclpy.create_node() 및 콜백(Callback) 함수#

  • 역할: ROS2에서 실행 가능한 노드 객체를 생성하는 파이썬 함수이다.

  • 사용법: node = rclpy.create_node('node_name') 형태로 노드 이름을 지정하여 생성

9.1.2.2. 콜백(Callback)#

  • 노드 내부에 콜백 함수를 등록하여 데이터를 처리한다.

  • 역할: Subscriber가 메시지를 받거나, 타이머가 만료되었을 때, 또는 서비스 요청을 받았을 때 비동기적으로 호출되어 실행되는 함수

  • 코드 예시:

def timer_callback(self):
    # 1초마다 이 코드가 실행됨 
    self.get_logger().info('Hello ROS2')

  • 노드가 동작하는 동안 ‘특정 로직을 수행’한다.

주로 AMCL (Adaptive Monte Carlo Localization)에서 위치 추정 방식으로 자율주행 로봇이 위치 추정 정보를 받아들이고 처리하는 핵심 함수이다. 주로 맵 상에서 로봇의 현재 위치 (Pose)가 업데이트 될 때마다 호출되는 콜백 함수를 지칭한다.

sub_amcl = rclpy.create_node('sub_amcl')

def amcl_callback(msg):
    print("===")
    print(msg.pose.pose.position)
    print(msg.pose.pose.orientation)

9.1.2.3. node.create_subscription()#

특정 토픽을 구독하고 메시지를 수신할 콜백 함수를 등록하는 함수

  • 기본 구조

self.subscription = self.create_subscription(MsgType, 'topic_name', self.callback, qos_profile)

  • 매개변수:

    • MsgType: 수신할 메시지 행식 지정 (예: std_msgs.msg.String)

    • ‘topic_name’: 구독할 토픽의 이름(문자열)

    • self.callback: 메시지를 받았을 때 실행할 함수

    • qos_profile: QoS (Quality of Serivce) 설정 (기본값: 10)

  • 용도: 토픽을 통해 발행된 메시지를 실시간으로 받아 명령이나 데이터를 처리하는데 사용

9.2. nav2_simple_commander#

ROS2 Navigation2(Nav2) 스택의 기능을 파이썬 스크립트에서 호출할 수 있도록 설계된 API 라이브러리이다. 액션 클라이언트(Action Client)나 ROS2 내부 작동 방식을 일일이 구현하지 않아도 로봇의 ‘네비게이션 기능’을 제어할 수 있게 해준다.

  • 로봇의 목적지 설정 및 제어를 가장 단순화하여 사용할 수 있는 라이브러리

  • 주요 용도

    • 파이썬 기반 네이게이션 제어

    • 웨이포인트(Waypoint) 내비게이션

    • 로봇 상태 모니터링

    • 간단한 작업 흐름 자동화

from nav2_simple_commander.robot_navigator import BasicNavigator 
import rclpy

rclpy.init()
nav = BasicNavigator() # Nav2 클래스 불러오기
