# 기술 조사 보고서 (Technical Research Report)

지금 단계에서, 기술조사 보고서 = “우리 시스템 설계를 뒷받침하는 근거 문서”이다. 기술 선택 이유를 설명하는 게 핵심이다.  즉, 기술 조사 보고서는 우리가 어떤 기술을 왜 선택했는지 설명하는 문서이다. 

아래는 간략히 기술 조서 보고서를 작성했고 상세 페이지에 세부 내용이 채워져있다. 

## 목적

본 보고서는 주조 공정 및 물류 시스템이 결합된 스마트팩토리 구현을 위해 필요한 핵심 기술을 조사하고, 적용 가능성을 분석하는 것을 목적으로 한다.

## 시스템 구성 요소

본 시스템은 다음 3가지 영역으로 구성된다:

- 주조 공정 (Casting)
- 물류 시스템 (AMR 기반 이송)
- 관제 시스템 (Control / Monitoring)

## 기술 조사 항목

### 주탕 자동화 기술 (Pouring Automation)

| 항목 | 내용 |
|------|------|
| 필요성 | 고온 / 위험 / 반복 작업 |
| 기술 | 산업용 로봇팔 (KUKA, UR 등) |
| 핵심 기능 | 정밀 위치 제어, 경로 계획 |
| 추가 기술 | 비전 기반 위치 인식 |
| 적용 난이도 | 중 ~ 높음 |
| 기대 효과 | 안전성 향상, 품질 균일화 |

### AMR (Autonomous Mobile Robot)

| 항목 | 내용 |
|------|------|
| 필요성 | 공정 간 자동 이송 |
| 기술 | ROS2 + Nav2 |
| 핵심 기능 | 자율주행, 경로 계획 |
| 센서 | LiDAR, Camera |
| 적용 난이도 | 중 |
| 기대 효과 | 유연한 물류 시스템 |


### Robot Manipulation

| 항목 | 내용 |
|------|------|
| 기술 | MoveIt2 |
| 기능 | Pick & Place, 경로 계획 |
| 제어 방식 | Inverse Kinematics |
| 적용 대상 | 주탕, 탈형, 후처리 |

### Vision System (검사 + 인식)

| 항목 | 내용 |
|------|------|
| 역할 | 위치 인식 / 결함 검사 |
| 기술 | OpenCV, YOLO |
| 기능 | Object Detection |
| 적용 위치 | 주탕 위치 정렬, 검사 |

### 관제 시스템 

| 항목 | 내용 |
|------|------|
| 역할 | 전체 시스템 관리 |
| 기술 | ROS2 Topic / Service / Action |
| 기능 | Task allocation, Monitoring |
| 확장 | LLM 기반 명령 해석 |

## 기술 선택 근거

- 주탕 공정은 고온 및 위험 작업이므로 로봇 자동화가 필수적임
- 물류 시스템은 다품종 소량 생산 환경에 대응하기 위해 AMR을 선택함
- 비전 시스템을 통해 공정 정밀도 및 품질 검사를 자동화함
- ROS2 기반 시스템을 통해 각 모듈 간 확장성과 유연성을 확보함

## 기존 시스템 vs 제안 시스템

| 항목 | 기존 | 제안 |
|------|------|------|
| 주탕 | 수작업 | 로봇 자동화 |
| 물류 | 수동/컨베이어 | AMR |
| 검사 | 육안 | Vision |
