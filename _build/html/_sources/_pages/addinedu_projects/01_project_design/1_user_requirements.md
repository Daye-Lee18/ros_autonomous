## Prerequisite 

### 하이브리드 스마트 팩토리 

하이브리드 스마트팩토리란,기존의 고정식 자동화(Fixed Automation)와 유연한 자동화(Flexible Automation)를 결합하여 운영하는 방식을 통칭한다. 쉽게 말해, "컨베이어 벨트처럼 정해진 길로만 빠르게 움직이는 설비"와 "AMR(자율주행로봇)처럼 상황에 따라 길을 바꾸는 설비"를 섞어서 쓰는 공장을 의미합니다.

| 구분 | 고정식 자동화 (Fixed) | 유연한 자동화 (Flexible) |
| ---|---|---|
|대표 설비 | 컨베이어 벨트, 전용 가공기 |AMR(자율주행로봇), 협동 로봇 |
| 장점 | 대량 생산 시 속도가 매우 빠름| 다품종 소량 생산에 유리, 경로 변경 쉬움|
| 단점 | 라인 변경이 어렵고 공간을 많이 차지함| 컨베이어보다 이동 속도가 느릴 수 있음|
| 역할 | 메인 도로 (공정 간 대량 수송)| 라스트 마일 (로봇팔 전달, 빈 공간 이송)|

### 영세 공장 현황

- 공장은 이미 기계적인 자동화가 되어 있지만, 중소형 공장은 사람의 손길이 필요한 구간(렉킹/검사)'과 '비용 문제' 때문에 완전 자동화에 어려움을 겪고 있는 상태 ->국가에서 자동화 전환 지원 사업 진행 중
- 다품종 소량 생산: 대기업 하청이 아닌 일반 공장은 일 들어오는 물건의 모양과 재질이 다름
- 이때마다 로봇 팔의 동선을 새로 짜는 것이 비용 대비 효율이 떨어져 수동을 고집하게 됨

### 실무 용어 

주조 분야에서는 용어가 중요하다. 신뢰도의 차이가 크기 때문이다. 

* 주탕(Pouring)
* 탈형 (Shakeout/Mold Removal)

## 주조 공장 자동화 

주조(Casting)란, 금속을 융점 이상의 고온으로 녹여 액체(용탕) 상태로 만든 뒤, 모래나 금속 등으로 만든 거푸집(주형)에 부어 식혀서 원하는 형상의 제품(주물)을 만드는 기초 제조 공정이다. 

### 시장 조사 및 현황 분석 (Market & Status)

* 국내외 현황: 
  * 국내 주조업계의 고령화 및 인력난 심화(3D 기피).
  * 글로벌 스마트 제조 시장의 성장세
* 기존 공정의 한계(Pain Points): 
  * 고온 용탕 주입 시 화상 위험.
  * 무거운 주물 이송 시 근골격계 질환 및 안전사고 발생.
  * 수동 작업으로 인한 품질 불균일.
  * 고온 작업 환경 및 분진

## [조사]주조 공정 전체 흐름 정리 

```
원재료 준비 → 용해 → 주형 제작 → 주입(주조) → 냉각 → 탈형 → 후처리 → 검사
```

## [분석] 실제 자동화가 필요한 작업


1) 원재료 준비
 * 사람이 하는 일
   * 원재료 선택 / 투입
   * 비율 조정
   * 품질 확인
 * 특징
   * 경험 기반 판단 많음
 * 자동화 판단
   * 가능: 투입, 계량 
   * 인간 필요: 소재 품질 판단
2) 용해 (Melting)
   * 사람이 하는 일
     * 온도 확인
     * 합금 조성 조절
     * 상태 판단 (색, 점도 등)
   * 자동화 판단
     * 가능: 온도 제어 (센서 + PID)
     * 인간 필요: 예외 상황 대응
3) 주형 제작 (Molding)
   * 사람이 하는 일
     * 모래 압축 / 금형 조립
     * 품질 확인 (균열, 정밀도)
   * 자동화 판단
     * 매우 쉬움 (이미 많이 자동화됨)
     * 인간 필요: 초기 셋업 / 오류 수정
4) 주입 (Pouring) ⭐ 핵심 위험 구간
    * 사람이 하는 일
      * 쇳물 붓기
      * 타이밍 조절
      * 흐름 제어
    * 자동화 판단 
      * 반드시 자동화해야 하는 구간
        * 고온 / 위험
        * 반복 작업
        * 로봇팔 + 비전 적용 가능
5) 냉각 / 탈형
   * 사람이 하는 일 
     * 타이밍 판단
     * 제품 꺼내기
   * 자동화 판단
     * 가능 (컨베이어 + 로봇)
     * 인간 필요: 비정상 제품 대응
6) 후처리 (절단, 연마 등)
   * 사람이 하는 일
     * 불필요 부분 제거
     * 표면 처리
   * 자동화 판단
     * 가능 (CNC / 로봇)
     * 인간 필요: 복잡 형상

7) 검사 (Inspection)
    * 사람이 하는 일
      * 외관 검사
      * 불량 판단
    * 자동화 판단
      * AI Vision 적용 가능
      * 인간 필요: 최종 판단 / 클레임 대응


사람 때문에 필요한 작업과 실제 필요한 작업 구분하고 전자의 경우, 자동화 타겟으로 삼았습니다. 예를 들어, 반복 작업 (투입, 이동, 단순 검사), 위험 작업 (주입, 고온 작업), 물리 노동 (운반, 절단)은 자동화 타겟으로 삼았습니다. 하지만, 공정 자체에서 필요한 작업은 자동화 어려움 or 보조 필요한데요. 이는 다음과 같습니다. 판단 (품질, 이상 상황), 최적화 (공정 조건 조절), 의사결정 (라인 변경) 부분은 AI와 인간이 협업을 해야하는 부분입니다. 


스마트팩토리는 단순 자동화가 아니라 “전체 공정을 연결하고 최적화”하는 것이고, 자동화되어도 사람은 사라지지 않습니다. 단지 역할이 작업자에서 감독자 / 분석가로 바뀝니다. 

결론적으로 자동화 우선순위는 다음과 같습니다. 

위 분석을 기반으로, 본 시스템은 "위험 작업의 자동화 + 판단 작업의 인간/AI 협업"을 핵심 설계 원칙으로 한다.

```yaml
* 1순위 (무조건 자동화)
    * 주입 (pouring)
    * 물류 이동 (AMR)
    * 단순 반복 작업
* 2순위 (AI + 자동화)
    * 검사 (vision)
    * 공정 최적화 (온도, 시간)
* 3순위 (사람 유지)
    * 예외 처리
    * 공정 설계
    * 품질 최종 판단
```

## User requirements 

* 목표: 다품종 소량 생산 환경에서 안전성과 효율성을 확보하는 하이브리드 스마트팩토리 데모 구축

* 사용자: 
  * 공장 작업자 (Operator)
  * 공정 관리자 (Manager)
  * 시스템 관리자 (Engineer)

* 기능 요구 (Functional Requirements):

  * 로봇팔:
    * 고온 주입(Pouring) 자동 수행
    * 주형 → 컨베이어 → 후처리 위치로 Pick & Place
    * 비전 기반 위치 인식 및 보정

  * AMR:
    * 공정 간 물류 자동 이송 (주형 → 주조 → 후처리)
    * 장애물 회피 및 동적 경로 계획
    * 작업 요청(Task) 기반 이동 수행

  * 검사 시스템:
    * 카메라 기반 외관 검사 (불량 탐지)
    * 결과를 중앙 시스템에 전달

  * 관제 시스템:
    * 작업 상태 모니터링 (로봇/AMR 상태)
    * 작업 지시 (Task Allocation)
    * 이상 상황 알림

* 비기능 요구 (Non-functional Requirements):

  * 처리 시간:
    * AMR 이동 지연 최소화 (실시간 경로 재계획)
  
  * 안정성:
    * 고온 작업에서 안전 확보 (로봇 대체)
  
  * 확장성:
    * 다양한 제품 형상 대응 가능 (다품종 생산)

  * 신뢰성:
    * 24시간 연속 운영 가능

* 시나리오: 
    1. 작업자가 원재료를 투입하거나 자동 공급 시스템이 투입
    2. 용해로에서 온도 센서 기반 상태 모니터링
    3. 로봇팔이 주탕(Pouring) 작업 수행
    4. AMR이 주형을 다음 공정으로 이송
    5. 로봇팔 또는 설비가 탈형 수행
    6. 후처리 공정 진행 (자동/반자동)
    7. 비전 검사 시스템이 불량 여부 판단
    8. 관제 시스템이 전체 상태를 모니터링 및 기록
