7.4.1. Prerequisite#

7.4.1.1. 하이브리드 스마트 팩토리#

하이브리드 스마트팩토리란, 기존의 고정식 자동화(Fixed Automation)와 유연한 자동화(Flexible Automation)를 결합하여 운영하는 방식을 의미한다. 쉽게 말해, “컨베이어 벨트처럼 정해진 경로로 빠르게 움직이는 설비”와 “AMR(자율주행 로봇)처럼 상황에 따라 경로를 변경하는 설비”를 함께 사용하는 공장을 의미한다.

구분

고정식 자동화 (Fixed)

유연한 자동화 (Flexible)

대표 설비

컨베이어 벨트, 전용 가공기

AMR(자율주행로봇), 협동 로봇

장점

대량 생산 시 속도가 매우 빠름

다품종 소량 생산에 유리, 경로 변경 쉬움

단점

라인 변경이 어렵고 공간을 많이 차지함

컨베이어보다 이동 속도가 느릴 수 있음

역할

메인 도로 (공정 간 대량 수송)

라스트 마일 (로봇팔 전달, 빈 공간 이송)

7.4.1.2. 영세 공장 현황#

  • 공장은 이미 일부 기계 자동화가 되어 있지만, 중소형 공장은 사람의 손길이 필요한 구간(렉킹/검사)과 비용 문제로 인해 완전 자동화에 어려움을 겪고 있음 → 국가 차원의 자동화 전환 지원 사업 진행 중

  • 다품종 소량 생산: 대기업 하청이 아닌 일반 공장은 일 들어오는 물건의 모양과 재질이 다름

  • 이때마다 로봇 팔의 동선을 새로 짜는 것이 비용 대비 효율이 떨어져 수동을 고집하게 됨

7.4.2. [조사] 주조 공장 자동화#

주조(Casting)란, 금속을 융점 이상의 고온으로 녹여 액체(용탕) 상태로 만든 뒤, 모래나 금속 등으로 만든 거푸집(주형)에 부어 식혀서 원하는 형상의 제품(주물)을 만드는 기초 제조 공정이다.

7.4.2.1. 시장 조사 및 현황 분석 (Market & Status)#

  • 국내외 현황:

    • 국내 주조업계의 고령화 및 인력난 심화(3D 기피).

    • 글로벌 스마트 제조 시장의 성장세

  • 기존 공정의 한계(Pain Points):

    • 고온 용탕 주입 시 화상 위험.

    • 무거운 주물 이송 시 근골격계 질환 및 안전사고 발생.

    • 수동 작업으로 인한 품질 불균일.

    • 고온 작업 환경 및 분진

7.4.3. [조사]주조 공정 전체 흐름 정리#

원재료 준비 → 용해 → 주형 제작 → 주탕(Pouring) → 냉각 → 탈형 → 후처리 → 검사

7.4.6. [결론] 자동화 우선순위#

본 공정 분석 결과, 주탕(Pouring) 공정은 고온 환경에서 수행되는 위험 작업이며 동시에 반복성이 높아 자동화 효과가 가장 큰 구간으로 판단된다. 따라서 본 시스템에서는 주탕 공정의 자동화를 최우선 목표로 설정한다.

본 시스템은 “위험 작업의 자동화 + 판단 작업의 인간/AI 협업”을 핵심 설계 원칙으로 한다. 즉, 사람 때문에 필요한 작업과 공정 자체에서 필요한 작업을 구분하고, 전자의 경우 자동화 타겟으로 설정하였다. 예를 들어, 반복 작업(투입, 이동, 단순 검사), 위험 작업(주탕, 고온 작업), 물리 노동(운반, 절단)은 자동화 대상으로 분류하였다. 반면, 공정 자체에서 필요한 작업은 완전 자동화가 어렵거나 보조가 필요한 영역으로 판단하였다. 이에 해당하는 작업은 다음과 같다: 판단(품질, 이상 상황), 최적화(공정 조건 조절), 의사결정(라인 변경). 이러한 영역은 AI와 인간의 협업이 필요한 부분이다.

* Priority 1 (Critical Automation)
    * 주탕 (Pouring) ← 핵심 자동화 대상
    * 물류 이동 (AMR)
    * 단순 반복 작업

* Priority 2 (AI-assisted Automation)
    * 검사 (Vision-based Inspection)
    * 공정 최적화 (온도, 시간)

* Priority 3 (Human-in-the-loop)
    * 예외 처리
    * 공정 설계
    * 품질 최종 판단

```yaml
* 1순위 (무조건 자동화)
    * 주탕 (pouring)
    * 물류 이동 (AMR)
    * 단순 반복 작업
* 2순위 (AI + 자동화)
    * 검사 (vision)
    * 공정 최적화 (온도, 시간)
* 3순위 (사람 유지)
    * 예외 처리
    * 공정 설계
    * 품질 최종 판단
